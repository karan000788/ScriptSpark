⚙️ Settings

🔑 API Keys Optional

Add free API keys to unlock higher-quality content. All keys are stored only in your browser (IndexedDB), never sent to any server.

Get your free key in 30 seconds at pexels.com/api · Sign up → Dashboard → API Keys. We use this to fetch real stock videos matching your topic as backgrounds.

🎤 Voice (Text-to-Speech)

Pick the voice that will read your script. Voices depend on your browser & OS. Install more voices: Windows: Settings → Time & Language → Language → Add Hindi/Tamil/etc → Speech. macOS: System Settings → Accessibility → Spoken Content → System Voice → Customize.

💾 Data

All your data is stored locally in your browser using IndexedDB. Nothing is sent to any server.

📁 My Projects

All your projects are auto-saved in your browser. Click any project to reload it.

How to use ScriptSpark 🚀

1. Pick your language (10 Indian languages supported).

2. Tell us your video topic/niche.

3. Write a brief about what the video should cover (this is the secret to great scripts).

4. Choose Shorts (≤60s) or Long Video + duration.

5. Pick a viral idea from 5 generated options.

6. Get a retention-optimized script + copyright-free music.

7. Edit scenes scene-by-scene — text, order, add, delete.

8. Render & download the video. Your project is auto-saved.
